Hygraph Tools

A suite of tools to help you manage and analyze your Hygraph content.

Included tools:

  • Schema Explorer - Analyze models, components, enums
  • Content Calendar - View scheduled content
  • Content Health - Find stale drafts & unused models
  • Alt-Text Generator - AI-powered image descriptions
  • Image Transform - Resize, optimize & generate responsive images
  • Content Analytics - Velocity, trends & author activity
Note: The SDK provides automatic authentication. If any tool asks for a PAT, enter it once and it will be shared across all tools.
